# Fuel Efficiency Prediction

A Machine Learning web application built using Streamlit and XGBoost to predict vehicle fuel efficiency (MPG).

## Features
- Predicts MPG based on vehicle specifications.
- Interactive Streamlit interface.
- Displays feature importance.
- Correlation heatmap.
- Actual vs Predicted comparison.

## Technologies Used
- Python
- Streamlit
- XGBoost
- Pandas
- Scikit-learn
- Joblib
